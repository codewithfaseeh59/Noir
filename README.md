# Noir — Fashion Brand Website

A modern, dark-themed fashion brand website built with vanilla HTML, CSS, and JavaScript. Features smooth animations, page transitions, and a fully responsive design.

## ✨ Features

- Dark/Light theme toggle with smooth transition
- SVG logo loader animation on page load
- Hero text split animation using Splitting.js + GSAP
- Scroll-triggered animations on all sections
- Collections filter by category (Evening, Casual, Luxury, Street)
- Real-time search in collections
- Sort by price (Low to High / High to Low)
- Lookbook horizontal swiper slider
- Custom animated cursor using SheryJS
- Hamburger menu with GSAP open/close animation
- Fully responsive (Mobile, Tablet, Laptop, Desktop)
- Statement section parallax effect
